# $QIET Token Utility

## Core Utilities
1. **Governance** - Vote on protocol parameters including pool fees
2. **Fee Distribution** - Protocol fees are distributed to $QIET holders
3. **Staking** (Planned) - Stake $QIET for enhanced relayer access

## Fee Mechanism
- Pool deposits and withdrawals incur a fee (set by governance)
- Fees accumulate in the protocol treasury
- Distributed proportionally to $QIET holders

## Future Utilities
- Cross-chain attestor staking
- Agent messaging fees in $QIET
- Escrow dispute resolution staking
