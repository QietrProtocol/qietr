# Qietr Docs - Welcome

## What is Qietr?
Qietr is a zero-knowledge privacy layer built on Solana for HTTP 402 micropayments.

- **x402**: Standard web micropayments - transparent, direct wallet-to-merchant payments
- **Qietr**: Privacy layer - anonymous pool-based payments with ZK proofs

Key Difference:
- **x402**: Alice's wallet -> Bob (merchant) | Everyone sees Alice paid Bob
- **Qietr**: Pool -> Bob (merchant) | Bob sees payment, can't identify Alice

## $QIET Token
- Revenue: Protocol fees distributed to token holders
- Governance: Vote on fees, features, upgrades
- Multi-product: Value from all Qietr products
- Network: Solana
