# $QIET Token Overview

## Summary
$QIET is the native governance and utility token of the Qietr protocol.

## Key Details
- **Network:** Solana (SPL token)
- **Role:** Governance, fee distribution, protocol incentives
- **Fee Share:** A portion of protocol fees flows to $QIET holders

## Governance
- Vote on pool fee parameters
- Vote on protocol upgrades and features
- Vote on verifying key upgrades (circuit updates)

## Distribution
Initial distribution focuses on:
- Community allocation
- Protocol development
- Liquidity provisioning
